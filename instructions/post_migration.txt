# План ускоренной миграции из Lotus в Exchange

  - **Определяем фронт работ:**
    - Выгружаем список пользователей Lotus, исключаем из списка уволенных и служебные ящики;
    - Выгружаем список пользователей доменов REM и NPR, насколько это возможно, отсеиваем дубликаты (оставляем их только в NPR);
    - Отсеиваем из списка NPR пользователей ЕКПС уже смигрированных в Exchange (как вариант  -по заполненному полю EmployeeId);
    - Сверяем список NPR со списком пользователей Lotus, определяем кто в NPR _не смигрирован_;
    - Сверяем список REM со списком пользователей Lotus.
    - Обогащаем список REM информацией о машинах пользователей и операционных системах (насколько возможно);
    - Готовим два списка по REM:
      - Пользователи с ПК готовыми к миграции в NPR (Windows 7);
      - Пользователи с ПК не готовыми к миграции в NPR (XP).
  - **Первый этап работ:**
    - УЗ всех пользователей Lotus кто в NPR _не смигрирован_,  готовим для миграции и подаём на миграцию (если такие вообще есть);
  - **Второй этап работ:**
    - УЗ всех пользователей Lotus по списку "Пользователи с ПК готовыми к миграции в NPR" приводим в соответствие с _ЕКП именования УЗ_, дополняем информацию для миграции в Exchange (готовые УЗ переносим в отдельный контейнер);
    - По списку "Пользователи с ПК готовыми к миграции в NPR" проводим миграцию в NPR (с переименованием ПК в соответствии с ЕКП);
    - Ежедневно проводим копирование подготовленных УЗ в NPR (из контейнера подготовленных УЗ) и подаём их на миграцию;
  - **Третий этап работ:**
    - По списку "Пользователи с ПК не готовыми к миграции в NPR" приводим УЗ в соответствие с _ЕКП именования УЗ_, дополняем информацию для миграции в Exchange (готовые УЗ переносим в отдельный контейнер);
    - Готовим инструкцию для пользователей "Как запустить MS Outlook от имени другого пользователя";
    - Распространяем MS Outlook;
    - Ежедневно проводим копирование подготовленных УЗ в NPR (из контейнера подготовленных УЗ) и подаём их на миграцию;
    - **Важно:** пользователей этого списка заводим в NPR с бесконечным паролем;
